# Tic Tac Toe 🎮

An advanced, terminal-based Tic Tac Toe game built with Python.

## Features

- **Colorized Terminal UI** — ANSI-colored board with box-drawing characters
- **Minimax AI** — Unbeatable AI with alpha-beta pruning on Hard mode
- **3 Difficulty Levels** — Easy, Medium, and Hard
- **Player vs Player** — Local 2-player mode
- **Choose Your Marker** — Play as X or O
- **Score Tracking** — Wins, losses, and draws across rounds
- **Animations** — Thinking spinner and typewriter text effects
- **Win Highlights** — Winning cells get highlighted

## How to Play

```bash
python tic_tac_toe.py
```

Use numbers 1-9 to place your mark:

```
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```

## Requirements

- Python 3.6+
- A terminal that supports ANSI colors
